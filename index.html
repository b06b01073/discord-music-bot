A Brief Guide

Prefix of command is ! (指令需要驚嘆號前綴)

  • Help

    <!help | !h>

    Description: Print the link of this page in discord channel.

  • Play music

    <!play | !p> <youtube link | keyword>

    Description: Play a song from Youtube.

    Example:

    1. !play https://www.youtube.com/watch?v=QF08nvtHHCY&ab_channel=Black%26White
    2. !play lofi music
    The first exapmle is search by link, the second one is search by keyword.

  • Skip

    <!skip | !s>

    Description: Skip currently playing music.

  • Quit

    <!quit | !q>

    Description: Make the bot leave the voice channel.

  • List

    <!list | !ls | !l>

    Description: Show current playlist.

  • Set

    !set <message> <response>

    Description: 設定回覆訊息，當設定完成後，下次在discord頻道輸入message時，bot將會回覆response

    Example:

    1. !set 嗨 你好
